# City Skyline

A project built as part of the **Responsive Web Design** certification from freeCodeCamp.

## Description
A city skyline created using only HTML and CSS.  
The project focuses on:
- CSS Variables
- Linear and repeating-linear gradients
- Positioning (relative and absolute)
- Flexbox layout

No images or JavaScript were used.

## Goal
Practice visual composition and advanced CSS techniques without relying on external assets.
